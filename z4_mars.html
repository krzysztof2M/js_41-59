1)kiedy powstał JavaScript (podaj datę).

JavaScript powstał w 1995 roku

2)dlaczego jest to język skryptowy

JavaScript jest językiem skryptowym, ponieważ jest interpretowany bezpośrednio przez przeglądarkę

3)gdzie jest wykonywany JS (klient lub serwer)

JavaScript jest wykonywany głównie po stronie klienta, czyli w przeglądarce internetowej, ale dzięki Node.js może być również uruchamiany po stronie serwera.

4)dlaczego jest JS to bezpieczne narzędzie

JavaScript jest bezpieczny, ponieważ ma ograniczony dostęp do zasobów komputera, na którym działa.

5)czy jest to język obiektowy , jeśli tak to dlaczego.

Tak, JavaScript jest językiem obiektowym, ponieważ korzysta z obiektów, metod i zdarzeń. Umożliwia definiowanie i manipulowanie obiektami za pomocą modelu DOM (Document Object Model).

6)wymień zalety JS:
Zaleta 1: Prosty w nauce i szybki do opanowania.
Zaleta 2: Powszechnie używany – w Internecie dostępne są miliony przykładów jego zastosowania.
Zaleta 3: Ogranicza obciążenie serwerów, ponieważ część operacji wykonywana jest po stronie klienta.
7)Framework to:

Framework to zestaw narzędzi, który ułatwia tworzenie aplikacji, zapewniając strukturę kodu oraz gotowe funkcje i biblioteki.

8)Do aplikacji mobilnych można użyć Framework

frameworki takie jak React Native pozwalają na tworzenie aplikacji mobilnych, które działają na różnych systemach operacyjnych.

9)Aplikacje responsywne to:

Aplikacje responsywne dostosowują się do rozmiaru ekranu, aby wyglądały i działały poprawnie na różnych urządzeniach. JavaScript pomaga w dynamicznym dostosowywaniu treści do rozdzielczości ekranu.

10. Wyjasnienie kodu: <input type="button" name="przycisk" value="Nowa Strona" onclick="WinOpen(' ')">

Ten fragment kodu tworzy przycisk, a po jego kliknięciu wywoływana jest funkcja WinOpen, która otwiera nową stronę (okno).

11. Wyjasnienie kodu: <window.open("obraz.html","okienko","toolbar=no,directories=no,menubar=no,height=280,width=160,top=200,left=200");>

Ten kod otwiera nowe okno o nazwie "okienko" z plikiem "obraz.html". Parametry określają m.in. ukrycie paska narzędzi (toolbar), katalogów (directories), paska menu (menubar) oraz wymiary i pozycję okna na ekranie.

12)Opisz następującą instrukcję window.close()

Instrukcja window.close() zamyka bieżące okno przeglądarki.
